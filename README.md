# Striver A2Z DSA Sheet Solutions

This repository contains my solutions to the **Striver A2Z DSA Sheet**, one of the most popular roadmaps for mastering Data Structures and Algorithms.

The goal of this repository is to improve problem-solving skills, strengthen DSA concepts, and prepare for technical interviews.

## 🚀 About the Sheet

The **Striver A2Z DSA Sheet** is a structured roadmap that covers all important DSA topics required for coding interviews.

Topics include:

- Arrays
- Strings
- Recursion
- Linked List
- Stack & Queue
- Binary Trees
- Binary Search Trees
- Graphs
- Dynamic Programming
- Greedy Algorithms
- Sliding Window
- Bit Manipulation
